PROJECT TITLE: Newsstand Register
PURPOSE OF PROJECT: 
To make a register for newspaper stand that can register different kinds of literature.
VERSION or DATE: 0.1
HOW TO START THIS PROJECT:
Create an object of the class register. 
AUTHORS: Oscar Wika, Thomas Todal, Kristoffer Martinsen & Siv Furland
USER INSTRUCTIONS:
To add a new newspaper call the addNewspaper method in the Register object.
If you wish to print out a list of all the newspapers in the Register call the showNewspapers method.
If you wish to remove a newspaper from the list you can call the removeNewspaper method.
